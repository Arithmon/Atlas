**Claim.** The E8xE8 heterotic string compactified on special-holonomy spaces
yields realistic four-dimensional gauge sectors; heterotic G2 backgrounds
with torsion require specific torsion classes to vanish.

**Relation.** Divergent, despite the shared E8xE8 vocabulary.

**Δ.** Heterotic G2 backgrounds require exactly the torsion component in
which GIFT's structure lives to vanish: the framework's near-solution sits in
pure Bryant class W2 (the 3-form is closed exactly, all residual torsion in
the coclosure), the opposite configuration. Arithmon borrows the E8xE8
architecture as motivation, not as string dynamics; nothing in the framework
assumes a string vacuum, and nothing in heterotic theory predicts this
geometry.
